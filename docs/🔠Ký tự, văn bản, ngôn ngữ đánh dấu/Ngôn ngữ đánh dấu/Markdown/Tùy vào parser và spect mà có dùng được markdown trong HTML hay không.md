Có thể trong `div` thì không được, nhưng trong `span` thì được. Nếu là CommonMark thì phải có xuống dòng chứ không được để sát:
```html
<div>

*Emphasized* text.

</div>
```

Hoặc dùng thuộc tính `markdown=1` trong thẻ HTML
