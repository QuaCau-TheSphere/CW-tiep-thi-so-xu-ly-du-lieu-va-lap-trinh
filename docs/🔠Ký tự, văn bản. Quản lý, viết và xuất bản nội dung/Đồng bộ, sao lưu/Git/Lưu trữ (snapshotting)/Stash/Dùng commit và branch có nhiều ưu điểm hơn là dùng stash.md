Khi nào mới cần dùng tới stash? Khi thoả cả 3 điều kiện sau:
- Index rối nùi và bạn cần phải chỉnh
- Working tree rối nùi và bạn cần phải chỉnh
- Bạn không có thời gian để chỉnh và phải tắt sớm
