title: ~ và ^ là để chỉ các commit trước đó
---
```
G   H   I   J
 \ /     \ /
  D   E   F
   \  |  / \
    \ | /   |
     \|/    |
      B     C
       \   /
        \ /
         A

A =      = A^0
B = A^   = A^1     = A~1
C = A^2
D = A^^  = A^1^1   = A~2
E = B^2  = A^^2
F = B^3  = A^^3
G = A^^^ = A^1^1^1 = A~3
H = D^2  = B^^2    = A^^^2  = A~2^2
I = F^   = B^3^    = A^^3^
J = F^2  = B^3^2   = A^^3^2
```


Lưu ý: `A~2 = D`, nhưng `A^2 = C`

> [!tip] Cách nhớ
> - `~` nằm ngang nên nó chỉ đi thẳng
> - `^` có sự hội tụ giữa hai đường nên nó là dành cho merge
