Nếu đang làm dở dang trên main rồi nhận ra từ đầu mình nên tạo branch mới thì dùng lệnh này:
```
git switch -c tênBranch
```

Trên VS Code thì cài plugin GitLens. Xong bấm <kbd>Ctrl+P</kbd>, chọn `GitLens: Git Switch To...`
